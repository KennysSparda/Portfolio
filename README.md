If you want run this project in your machine do:

  Download the files:
  ```git clone https://github.com/KennysSparda/Portfolio.git```

  Install Requiriments:
  ```npm install next react react-dev```

  Install aditional node modules:
  ```npm install```

  And run with:
  ```npm run dev```
